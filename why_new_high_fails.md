## 🔍 为什么"半年新高"如此致命？

### 原因1：过滤掉了大部分交易机会

你的策略本质是"突破买入"，而"半年新高"要求：
- 价格必须是半年内最高点

**但实际走势：**
```
好的买入点往往不是半年新高：

场景A：回调后第二波启动
    ↑
    │  ╱  半年高点
    │ ╱
    │╱  回调
    │╲
    │ ╲ 你的买入点（不是新高，但很好）
    │  ╲
    └──────────→

场景B：震荡突破
    ↑
    │    ╱──╲ 半年高点
    │   ╱    ╲
    │  ╱      ╲  多次触及高点
    │ ╱   买入点 ╲（放量突破，但不是半年新高）
    │╱____________╲
    └──────────→

场景C：V型反转
    ↑
    │              ╱ 半年高点
    │             ╱
    │    买入点  ╱（底部放量，但离半年高点很远）
    │   ___╱    
    │  ╱
    │_╱
    └──────────→
```

**结论**：
- 只有极少数交易能同时满足"放量阳线"+"半年新高"
- 大部分好的买入点都被过滤掉了
- 交易次数从53笔降到7笔，机会成本太高

### 原因2：幸存者偏差

能创半年新高的股票：
- 已经涨了很多
- 往往是行情末期
- 追高风险大

**而你策略赚钱的逻辑是：**
- 在启动初期买入
- 吃完整段趋势
- 半年新高时可能已经涨了80%，只剩20%空间

### 原因3：时间窗口太长

120天（半年）对于60分钟策略来说太长：
- 60分钟策略持仓周期：几天到几周
- 半年新高要求：过去120天的最高点
- 时间框架严重不匹配

## 💡 更合理的过滤条件

既然"半年新高"太严格，试试这些：

### 方案1：缩短时间窗口

```python
# 不是半年新高，而是近20天新高
lookback = 20  # 而不是120
recent_high = data['High'].rolling(lookback).max()
if close_price >= recent_high * 0.98:  # 允许2%误差
    allow_buy = True
```

**预期效果**：
- 保留更多交易机会
- 同时避免在绝对高点买入

### 方案2：创近期新高但不创历史高点

```python
# 创20天新高，但不是60天新高
# 捕捉第二波启动，而不是第一波末期
recent_20_high = data['High'].rolling(20).max()
recent_60_high = data['High'].rolling(60).max()

if close_price >= recent_20_high * 0.99 and close_price < recent_60_high * 0.95:
    allow_buy = True
```

### 方案3：相对位置过滤（不要绝对新高）

```python
# 价格在近期区间的上半部分
recent_low = data['Low'].rolling(20).min()
recent_high = data['High'].rolling(20).max()
position = (close_price - recent_low) / (recent_high - recent_low)

if position > 0.6:  # 在20天区间的上40%
    allow_buy = True
```

### 方案4：排除极端高位（反向操作）

```python
# 不追求新高，只排除极端高位
# 如果距离60天高点只有2%，说明涨太多了，不追
recent_60_high = data['High'].rolling(60).max()
if close_price < recent_60_high * 0.95:  # 距离高点至少有5%空间
    allow_buy = True
```

### 方案5：结合ATR的波动空间

```python
# 确保上方还有空间
atr = data['ATR'].rolling(14).mean()
recent_high = data['High'].rolling(20).max()

# 距离近期高点还有至少1个ATR的空间
if (recent_high - close_price) > atr:
    allow_buy = True
```

## 🎯 核心结论

**"半年新高"过滤失败的原因：**
1. 时间窗口太长（120天 vs 策略持仓几天）
2. 过滤掉太多有效交易（交易次数-87%）
3. 幸存者偏差（半年新高的股票往往已涨完）

**建议：**
- 放弃"半年新高"条件
- 如果要用，缩短到20天
- 或者反过来用"不追高"逻辑（排除极端高位）

**当前最优配置：**
- 基础策略 + 跳空保护 + 时间止损20天 + 分级止盈
- 这已经是一个较好的平衡点
