# us_stock 使用说明文档

本项目当前以 A 股策略扫描/回测为主，核心流程集中在以下目录：

- `scanners/`：选股与实时扫描
- `backtests/`：策略回测
- `data/`：数据拉取与分钟数据维护
- `scripts/`：结果图表和辅助工具
- `results/`：输出结果

## 1. 运行环境

- Python 3.10+（建议 3.10/3.11）
- Windows / macOS / Linux 均可（以下示例默认在项目根目录执行）

安装依赖：

```bash
python -m pip install -r requirements.txt
python -m pip install akshare
```

建议先验证脚本可用：

```bash
python scanners/scan_ma_convergence_daily.py --help
python backtests/backtest_ma_convergence.py --help
python scanners/scan_volume_breakout_realtime.py --help
python data/update_a_stock_minute.py --help
```

## 2. 目录与数据约定

### 2.1 日线缓存

- 默认目录：`data_cache/`
- 文件命名：`{symbol}_{years}y_1d_forward.csv`
- 示例：`600519.SS_20y_1d_forward.csv`

### 2.2 分钟缓存

- 默认目录：`data_cache/a_stock_minute/`
- 文件命名：`{symbol}_{period}min.csv`
- 示例：`000001.SZ_15min.csv`

## 3. 快速开始（推荐流程）

### 步骤 1：运行日线扫描（均线收敛）

默认股票池：`scanners/稳定行业.txt`

```bash
python scanners/scan_ma_convergence_daily.py --a-share-only --years 20 --workers 8
```

说明：

- 脚本默认会尝试补齐缺失缓存并刷新陈旧缓存（`--update-missing` 与 `--update-existing` 默认开启）
- 输出目录默认：`results/ma_convergence_daily_scan/`

常见扩展：

```bash
# 使用自定义股票池 CSV（symbol 列）
python scanners/scan_ma_convergence_daily.py --stock-pool-file scanners/stock_pool.csv --a-share-only --years 20

# 最近3根K线内有信号都输出
python scanners/scan_ma_convergence_daily.py --lookback-bars 3 --a-share-only --years 20

# 关闭止损 + 设置时间退出
python scanners/scan_ma_convergence_daily.py --no-stop-loss --time-exit-days 90 --a-share-only --years 20
```

### 步骤 2：对同一股票池做回测

```bash
python backtests/backtest_ma_convergence.py --a-share-only --years 20 --workers 8
```

输出目录默认：`results/ma_convergence_backtest/`

常见扩展：

```bash
# 仅回测指定股票池文件
python backtests/backtest_ma_convergence.py --symbols-file scanners/稳定行业.txt --workers 8

# 指定日期区间
python backtests/backtest_ma_convergence.py --start-date 2019-01-01 --end-date 2025-12-31 --workers 8

# 调整止损/成交量过滤参数
python backtests/backtest_ma_convergence.py --stop-loss-pct 0.15 --volume-filter-mode contraction --vol-ratio-max 0.8
```

### 步骤 3：为扫描结果生成图表（可选）

```bash
python scripts/generate_ma_convergence_charts_from_scan.py --scan-csv <扫描结果CSV路径> --data-years 20 --plot-years 5 --workers 4
```

输出通常在 `results/ma_convergence_scan_charts/`，包含：

- `charts.csv`
- `index.html`
- 多张个股 K 线 PNG

## 4. 缠论实时扫描（独立流程）

运行：

```bash
python scanners/scan_signals_realtime.py
```

输出目录：`results/realtime_scan/`

- `signals_history.csv`：历史信号累积
- `summary_latest.json`：最近一次扫描摘要

注意：

- 该脚本内部默认使用代理 `http://127.0.0.1:7897` 拉取数据；如果本机没有该代理，需要在脚本里修改 `DataFetcher(...)` 的 `proxy` 参数（文件：`scanners/scan_signals_realtime.py`）。

## 5. 分钟级数据与量能突破实时扫描（独立流程）

### 5.1 首次准备分钟数据

```bash
python data/fetch_a_stock_minute.py
```

该脚本会根据 `data_cache/` 里已有的 A 股日线缓存，生成分钟数据到 `data_cache/a_stock_minute/`。

### 5.2 每日增量更新分钟数据

单次执行：

```bash
python data/update_a_stock_minute.py --adjust qfq
```

定时模式（每天 17:10）：

```bash
python data/update_a_stock_minute.py --daemon --run-time 17:10 --adjust qfq
```

### 5.3 运行量能突破实时扫描

单次扫描：

```bash
python scanners/scan_volume_breakout_realtime.py --once --period 15
```

交易时段循环扫描（09:30-11:30, 13:00-15:00）：

```bash
python scanners/scan_volume_breakout_realtime.py --daemon --scan-interval-minutes 20 --period 15
```

启用模拟持仓管理：

```bash
python scanners/scan_volume_breakout_realtime.py --daemon --paper-trade --fixed-shares 100
```

输出目录默认：`results/volume_breakout_realtime/`

## 6. 常见问题

### 6.1 `ModuleNotFoundError: No module named 'data_fetcher'`

这是旧示例脚本路径问题。`examples/` 和 `scripts/main.py` 使用了历史导入方式，当前请优先使用本文档中的 `scanners/`、`backtests/`、`data/` 命令。

### 6.2 扫描提示没有可用 symbol

检查：

- `scanners/stock_pool.csv` 是否存在且 `enabled=1`
- `data_cache/a_stock_minute/` 下是否已有对应分钟文件
- symbol 格式是否为 `000001.SZ` / `600519.SS`

### 6.3 网络/数据拉取失败

- 可通过命令行参数传代理（例如 `--proxy http://127.0.0.1:7897`，适用于支持该参数的脚本）
- 或先离线准备缓存后再运行扫描/回测

## 7. 推荐日常执行顺序

1. 收盘后更新分钟数据：`python data/update_a_stock_minute.py --adjust qfq`
2. 跑日线扫描：`python scanners/scan_ma_convergence_daily.py --a-share-only --years 20 --workers 8`
3. 盘中跑量能突破扫描：`python scanners/scan_volume_breakout_realtime.py --daemon --paper-trade`
4. 周期性回测验证：`python backtests/backtest_ma_convergence.py --a-share-only --years 20 --workers 8`
